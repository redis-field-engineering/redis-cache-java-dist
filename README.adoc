== Quick Start

=== 1. Start Infrastructure

[source,console]
----
cd infrastructure
docker compose up
----

This starts Redis, Prometheus, and Grafana.

=== 2. Set TMDB Token

Register and create an API read-access token at https://developer.themoviedb.org/docs/getting-started[themoviedb.org].

[source,console]
----
export TMDB_TOKEN=<your API read-access token>
----

=== 3. Run the Demo

[source,console]
----
cd demo
./gradlew bootRun
----

=== 4. Access the Application

* **Demo App**: http://localhost:8080
* **Grafana Dashboard**: http://localhost:3000/d/1 (username/password: `admin`/`admin`)

See link:demo/README.md[demo/README.md] for detailed instructions.
